    AllowUserResizing

    Gets or sets a value indicating whether this form can be resized by the user. See remarks.

    Remarks

    This property alters System.Windows.Forms.Form.FormBorderStyle, for true value it is System.Windows.Forms.FormBorderStyle.Sizable, for false - System.Windows.Forms.FormBorderStyle.FixedSingle.
