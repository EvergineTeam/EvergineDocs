    Depth-first in-order traversal as described in http://en.wikipedia.org/wiki/Tree_traversal The only difference is using fixed size array instead of stack for speed-up. Note: By passing parentsStack you may reuse the stack array between different method calls, but it should be at least Height length. The contents of array are not important.
