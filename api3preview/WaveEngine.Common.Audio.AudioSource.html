    QueuePlayPosition

    Gets or sets the playback position relative to the beginning of the PendingBuffers queue.

    Remarks

    QueuePosition is ranging from System.TimeSpan.Zero to QueueDuration. Values beyond this range are clamped. When the audio source is in Stopped state and the value is higher than System.TimeSpan.Zero the state will change to Paused. The position is relative to the beginning of all the queued buffers for the source, and any queued buffers traversed by a set value will raise the event BufferEnded.
