    GetMetaExpressionOrDefault(Request)

    Discovers and combines service with its setup metadata. Works with any generic type with first Type arg - Service type and second Type arg - Metadata type, and constructor with Service and Metadata arguments respectively.

    • if service key is not specified in request then method will search for all registered factories with the same metadata type ignoring keys.
    • if metadata is IDictionary{string, object}, then the First value matching the TMetadata type will be returned.
