    Swap<T>(ref T, Func<T, T>, Int32)

    First, it evaluates new value using getNewValue function. Second, it checks that original value is not changed. If it is changed it will retry first step, otherwise it assigns new value and returns original (the one used for getNewValue).

    Remarks

    Important: getNewValue May be called multiple times to retry update with value concurrently changed by other code.
