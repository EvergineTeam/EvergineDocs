    Properties

    Summary

    Name Description
    AlphaTolerance

    Gets or sets alpha (coverage) tolerance. Default is 20: Every pixel with a coverage value equal or greater to 20 will be counts as solid.
    HoleDetection

    Gets or sets a value indicating whether will detect texture 'holes' if set to true. Slows down the detection. Default is false.
    HullTolerance

    Gets or sets default is 1.5f.
    MultipartDetection

    Gets or sets a value indicating whether will detect texture multiple 'solid' isles if set to true. Slows down the detection. Default is false.
    PixelOffsetOptimization

    Gets or sets a value indicating whether will optimize the vertex positions along the interpolated normal between two edges about a half pixel (post processing). Default is false.
    PolygonDetectionType

    Gets or sets get or set the polygon detection type.
    Transform

    Gets or sets can be used for scaling.
