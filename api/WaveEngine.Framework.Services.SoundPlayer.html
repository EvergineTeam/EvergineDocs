    Class SoundPlayer

    This class allows to play sounds and manages the active sounds.

    Inheritance
    System.Object
    SerializableObject
    Service
    UpdatableService
    Namespace:System.Dynamic.ExpandoObject
    Assembly:WaveEngine.Framework.dll
    public sealed class SoundPlayer : UpdatableService, IDisposable
    Remarks

    In Android, a SoundPool is used internally. This has a file size limit of 1MB for WAV sounds. Compressed files (MP3, OGG) are converted to PCM before playing, so they are affected by this limitation too.
