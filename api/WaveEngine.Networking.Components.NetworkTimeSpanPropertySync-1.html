    Class NetworkTimeSpanPropertySync<K>

    Gets or sets a value indicating whether the System.TimeSpan will be stored using high precision (ticks) or not (milliseconds)

    Remarks

    High precision allows full System.TimeSpan range, otherwise it will be limited to "+-24.20:31:23.647" (+-24 days, 20 hours, 31 minutes, 23 seconds and 647 milliseconds)
