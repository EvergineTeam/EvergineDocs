    Struct DX12MeshShaderPipelineStateStream

    Represents a DirectX 12 pipeline state stream description specifically tailored for Mesh Shaders. Each field corresponds to a subobject in the pipeline state stream. This structure must be passed when creating a pipeline state object (PSO) for a Mesh Shader pipeline.
