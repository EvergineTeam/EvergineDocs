    Class IFCColorHelpers

    Helper class for extracting and generating color information from IFC entities. Provides methods to retrieve colors from styled items, products, tessellated face sets, and to generate smooth vertex colors.

    GetColorByStyleLabel(int, IfcStore)

    Retrieves the color associated with a given style label from the IFC model. Attempts to extract the color from an IIfcSurfaceStyle or, if not found, from an IIfcPresentationStyleAssignment that contains the surface style. If no valid color is found, returns White as a fallback. The alpha channel is computed based on the transparency value of the surface style rendering.

    GetMaterialIndexFromStyle(int, IfcStore)

    Determines the material index (opaque or transparent) from a style label in the IFC model. Inspects the style entity referenced by the label, checks for surface style rendering, and evaluates transparency. Returns OPAQUE_INDEX if no rendering information is found or if transparency is less than or equal to 0.5. Returns TRANSPARENT_INDEX if transparency is greater than 0.5.
