    Class USDRuntime

    Represents the runtime for handling USD (Universal Scene Description) files in Evergine. This class provides methods to read, parse, and process USD files, including their materials, nodes, and geometries.

    ReadTexture(string, bool)

    Reads a texture from the specified image path. This method supports both standalone USD files and USDZ archives.

    Declaration
    public Task<Texture> ReadTexture(string imagePath, bool isGamma)
    Parameters
    Type Name Description
    string imagePath

    The relative or absolute path to the image file.
    bool isGamma

    Specifies if the texture has gamma correction.
    Returns
    Type Description
    Task<Texture>

    A task representing the asynchronous operation. The result contains the loaded Texture object, or null if the texture could not be loaded.
    Remarks
    • If the file extension is ".usdz", the method attempts to locate and read the texture from within the USDZ archive.
    • If the file is not part of a USDZ archive, it resolves the path relative to the working directory and loads the texture.
    • The method ensures that the image stream is properly disposed after reading.
