    Class GeometryHelper.SplineBuilder

    Utility class to build and evaluate a non-rational B-Spline (NURBS without weights).

    Remarks

    Expected invariants:

    • Degree (p) must be >= 1.
    • ControlPoints.Count must be >= p + 1.
    • Knots.Count must be == ControlPoints.Count + Degree + 1.
    • Knot vector must be non-decreasing. Typical uniform clamped knot vector example for n+1 control points and degree p: [0,...,0, internal increasing values ...,1,...,1] with (p+1) zeros and (p+1) ones at the ends. Algorithm:
    • Point evaluation uses the De Boor algorithm (stable form of de Casteljau analogue for B-Splines).
    • Public parameter t is normalized in [0,1]; it's mapped to the valid internal knot range [U[p], U[m-p-1]].
    • PolygonalVertexes produces a polyline sampling the spline (uniform parameter sampling, not arc-length).
